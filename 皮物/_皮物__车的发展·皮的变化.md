# [皮物]_车的发展·皮的变化



## 摘要  { data-search-exclude }

<!-- tcd_abstract -->
本文件为一篇以人皮和汽车发展为主线的文学作品，通过将人皮（拟人化的概念）与汽车的发展进行对比，探讨了技术与社会的变迁。文中描述了不同历史时期的汽车特点及人皮的进步，并通过作者的自我认知，呈现了从1967年到2017年的时代变迁。作者通过幽默和讽刺的语言，展现了对中国汽车工业和人皮技术发展的自豪感，并融入了对自我性别身份的探索。文中有如下精彩摘录：“香车美女，我们都爱。车子可以花钱买，美女却……不对，我们可以变成美女，不是吗？”这句反映了性别意识的转换与自我认同的冲突。

――――――――――

川流不息，车子搭载着我们的身体；变幻莫测，人皮包裹着我们的灵魂。汽车见证着祖

国的发展，皮物又何尝不是呢？

――题记



一九六七



随着一声清脆的“咔嚓”，我把档位切换到最高档位――三档。军绿色的北京212吉普怒吼着

把速度提高到70km/h。望着窗外飞速掠过的荒凉边境，我长叹一声。



九十，就是最高时速了吗？



擦的锃亮的金属仪表盘不能映出我复杂的心情，巨大的噪音也掩盖了我的叹息。



水箱开锅了！



我不情愿地停下了车。寒风吹到我脸上，我却并不寒冷，是那厚厚的人皮替我阻挡了风

寒。我摘下了墨镜，用呆滞的塑料眼望着发动机舱。



我们有了人皮，这没错。但是高昂的造价和粗劣的做工限制了它或“她”的发展。就像面

前这台212一样，只能应用于军队。或者说，军队的需求也只是勉强满足吧！



我身上的人皮，只解决了有无问题。“她”，在被设计出来的那一刻起就已经过时了。



一九八七



大众桑塔纳在川流不息的车潮中是那么显眼，就像穿着摩登的我。



宽大的车身，繁多的按键，“豪华”的氛围。



人皮早已民用化，虽然借用了外国科技，而且成本仍然高昂的只能被富人接受，就像这

台桑塔纳一样，但仍挡不住我们对“她”的热爱。



望着经济特区深夜的灯红酒绿，我的心也跟着迷惘起来。



拉开车门，我晃动着金色的大波浪，轻轻倚在车上，让车标从修长的双腿间露出，吸引

着众人的目光。



虽然没有变声器的人皮让我无法说话，也无法传达表情，但现在，我就是那个迷人的她

！



二零一七



化妆镜里的我笑靥迷人，朱红的嘴唇微张。轻踩油门，唯一能告诉我时速过百的东西是

迈速表，一起都是那么静谧，在这台沃尔沃里，仿佛只有我和“她”。



高水准的车，高水准的皮。



中国，在一天天地强大。我可以自豪地说：我们拥有不差于任何一个国家的车，还有人

皮。中国，我为你自豪！



小穴的振动打乱了我的思绪。



啊~啊~



我俏脸一红，双腿一紧。人皮联动系统太巧妙了，让我欲仙欲死。



虽然我处于迷离的高潮状态，但开启了自适应巡航的沃尔沃仍然在平稳地行驶，好像什

么也没发生一样。



香车美女，我们都爱。车子可以花钱买，美女却……



不对，我们可以变成美女，不是吗？



汽车，记录着社会的发展；人皮，见证着时代的变迁。



我们的祖国，将会庄严地宣誓：不管在人皮、汽车或是任何其他方面，中国，都会让世

界刮目相看！中国，我为你而感到自豪！

――――――――――
